✅ 1. PROGRAMACIÓN PARALELA (Criterio MÁS IMPORTANTE)
────────────────────────────────────────────────────────
   📁 Archivo: engine/SearchEngine.cpp
   
   Funciones Paralelas Implementadas:
   • buildTrigramIndexParallel()     → Trigramas en paralelo
   • buildWordIndexParallel()        → Índice invertido paralelo
   • buildTagIndexParallel()         → Tags en paralelo
   • buildIndexesParallel()          → Construye los 3 simultáneamente
   
   Características:
   • Usa std::thread nativo de C++
   • Mutex para sincronización
   • Particionamiento por hardware_concurrency()
   • Local aggregation para reducir contención
   
   Speedup Esperado: ~2.7x en construcción de índices


✅ 2. PATRÓN SINGLETON 🔒
────────────────────────────────────────────────────────
   📁 Archivo: engine/SearchEngine.h
   
   Implementación:
   • Instancia única global
   • Thread-safe con double-checked locking
   • getInstance() como punto de acceso
   
   Uso:
   SearchEngine* engine = SearchEngine::getInstance();


✅ 3. PATRÓN FACTORY METHOD 🏭
────────────────────────────────────────────────────────
   📁 Archivo: patterns/IndexFactory.h
   
   Implementación:
   • Crea índices de tipo Trigram o Inverted
   • Interfaz base Index<T>
   • Extensible para nuevos tipos
   
   Uso:
   auto index = IndexFactory<uint32_t>::createIndex(
       IndexFactory::IndexType::TRIGRAM
   );


✅ 4. PATRÓN BUILDER 🔨
────────────────────────────────────────────────────────
   📁 Archivo: patterns/QueryBuilder.h
   
   Implementación:
   • API fluida para construir queries
   • Configuración paso a paso
   • Validación integrada
   
   Uso:
   SearchQuery query = QueryBuilder()
       .setText("star wars")
       .asPhrase()
       .addTag("sci-fi")
       .setLimit(10)
       .build();


✅ 5. PATRÓN OBSERVER 👁️
────────────────────────────────────────────────────────
   📁 Archivo: patterns/MovieObserver.h
   
   Implementación:
   • MovieSubject como sujeto observable
   • ConsoleLogger para logs
   • StatisticsTracker para métricas
   
   Uso:
   MovieSubject subject;
   subject.attach(&logger);
   user.toggleLike(id);  // → notifica automáticamente
